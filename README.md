# Geek Updater

A Python script that automatically downloads and updates [**Geek Uninstaller**](https://geekuninstaller.com/) to the latest version, then launches it.

## Features
- Checks the official Geek Uninstaller website for the latest version
- Downloads and extracts the newest release
- Replaces the old `geek.exe` with the updated one
- Cleans up temporary files
- Launches Geek Uninstaller automatically after update

## Requirements
- Python 3.8+
- Dependencies listed in `requirements.txt`

Install dependencies:
```bash
pip install -r requirements.txt
```

# Usage
```bash
git clone https://github.com/yourname/geek-updater.git
cd geek-updater
python updater.py
```
